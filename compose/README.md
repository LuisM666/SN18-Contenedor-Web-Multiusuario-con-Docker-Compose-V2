# SN18 – Contenedor Web Multiusuario con Docker Compose

Proyecto académico ESIT – Proyecto 18  
Equipo SN-18

## Descripción
Implementación de un entorno de hosting web multiusuario utilizando Docker Compose, con múltiples sitios web desplegados en contenedores independientes sobre una máquina virtual en la nube.

## Alcance
- Dos sitios web simulados (site1 y site2)
- Aislamiento lógico entre contenedores
- Persistencia de datos mediante volúmenes Docker
- Gestión centralizada con Docker Compose
- Pruebas de acceso, aislamiento y persistencia documentadas (Fase 3)

## Estructura del repositorio
- `/compose` – Archivo docker-compose.yml
- `/sites` – Contenido web de cada sitio
- `/docs` – Documentación por fases (Fase 0 a Fase 3)
